/*
 * OVERVIEW
 */

Kontikiパッケージ（以下kontiki）はfuelphpのmoduleを基礎にしています。

kontikiを用いて構築する際は、原則PKGPATH.'kontiki/modules/'以下に、以下のような構成のファイルをおいてください。

modulename
- classes
	- controller
		- modulename.php
	- model
		- modulename.php
	- view
		- modulename.php
- migrations
- views

kontikiには、基本的なcrudを実装した基礎classがあります。この基礎classは、fuelのclassを継承していますが、必ず使わなくていけないということはありません。必要に応じてfuelのclassを直に継承して問題はありません。



/*
 * APPPATH.config/config.php
 */

- always_loadに、kontikiをたす

- modulesを以下のように設定
		 'modules'  => array(
		 	'acl',
		 	'user',
		 	'usergroup',
		 ),


- Security settingsの
	'csrf_autoload'    => false,
	'csrf_token_key'   => 'fuel_csrf_token',
	'csrf_expiration'  => 0,
	コメントアウトを外す

/*
 * APPPATH.config/routes.php
 */

- トップページと404を変更
	'_root_'  => 'content/home',
	'_404_'   => 'content/404',
